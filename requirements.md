# 视频知识脑图平台 - 前端 Demo 需求文档

## 项目定位

高保真视觉原型，展示核心界面布局与交互逻辑，按钮与控件全部呈现，功能以模拟数据演示为主，无需真实后端支撑。

---

## 一、整体布局

- **设备**：桌面端（PC）
- **结构**：左右分屏，固定视口高度，无需滚动
- **风格**：大气、简洁、高级；深色主调，细节精致

```
┌─────────────────────────┬──────────────────────┐
│                         │                      │
│     视频播放区           │                      │
│                         │     脑图区            │
├─────────────────────────┤                      │
│                         │                      │
│     弹幕 / 评论区        │                      │
│                         │                      │
└─────────────────────────┴──────────────────────┘
         左侧 ~55%                右侧 ~45%
```

---

## 二、左侧区域

### 2.1 视频播放区（上）

| 元素 | 说明 |
|------|------|
| 视频画面 | 占位画面（深色背景 + 居中播放图标） |
| 进度条 | 可视化时间轴，标记弹幕时间点 |
| 控制栏 | 播放/暂停、快进/快退、音量、倍速、全屏 |
| 时间显示 | 当前时间 / 总时长 |
| 章节标题 | 当前播放章节名称显示在播放区上方 |

### 2.2 弹幕 / 评论区（下）

#### 弹幕展示列表

- 评论按时间点排列，显示：
  - 用户头像 + 昵称
  - 绑定时间点（如 `02:34`，点击可跳转视频）
  - 评论内容
  - 点赞数

- 当视频"播放"到对应时间点，该评论高亮显示（模拟回放弹幕出现效果）

#### 发送评论栏

- 输入框：`在此输入你的想法…`
- 显示当前绑定时间（自动取视频当前时刻）
- 发送按钮

---

## 三、右侧区域

### 3.1 脑图区

| 元素 | 说明 |
|------|------|
| 标题栏 | "知识脑图" + AI 生成标签 |
| 脑图主体 | 树状/放射状节点图，展示视频核心知识点（静态模拟数据） |
| 节点样式 | 中心节点突出，子节点分层展示，连线细腻 |
| 管理员编辑控件 | 编辑按钮（铅笔图标）、新增节点、删除节点、保存（普通用户隐藏或置灰） |
| AI 生成按钮 | "AI 萃取知识点" 按钮，点击后显示 loading 动画（模拟） |

## 四、UI 风格规范

### 配色

| 用途 | 色值 |
|------|------|
| 背景主色 | `#0D0F14`（深黑） |
| 面板背景 | `#141720` |
| 边框/分割线 | `#252A35` |
| 主强调色 | `#4F8EF7`（蓝） |
| 辅助强调色 | `#A78BFA`（紫） |
| 文字主色 | `#E8ECF4` |
| 文字次色 | `#6B7280` |

### 字体

- 中文：系统默认无衬线
- 英文/数字：等宽或现代感无衬线
- 层级：标题 18px / 正文 14px / 辅助 12px

### 交互风格（视觉层面）

- 按钮 Hover：轻微发光或背景加亮
- 卡片：圆角 `8px`，轻微阴影
- 动效：过渡自然，不过度

---

## 五、模拟数据说明

- 视频：占位黑色画面，进度条预设总时长 `45:00`
- 弹幕：预置 6-8 条分布在不同时间点的评论
- 脑图：预置一套 2 层知识点结构（主题 → 子主题 → 知识点）
- 用户身份：页面右上角有"管理员/普通用户"切换按钮，影响脑图编辑控件的显示

---

## 六、开发步骤规划

1. **Step 1**：搭建整体布局框架（左右分屏 + 头部导航）
2. **Step 2**：实现视频播放区视觉（控制栏 + 进度条 + 时间点标记）
3. **Step 3**：实现弹幕 / 评论区（列表 + 时间点高亮 + 发送栏）
4. **Step 4**：实现脑图区（节点渲染 + 编辑控件）
5. **Step 5**：打磨细节与 UI 一致性
